## Overview

**Grama Suvidha Portal** is a smart Android application designed to simplify access to rural governance services.
It allows users to submit complaints, provide feedback, and interact with administrative services through a clean and user-friendly mobile interface.

This project is developed as part of an academic/internship program using **Kotlin and Android Studio**.

---

## Objective

The main goal of this project is to digitize basic village-level services and improve communication between citizens and administration through a mobile platform.

---

##  Key Features

### User Features

*  Home Dashboard for navigation
*  Submit complaints easily
*  Send feedback to administration
*  Simple and responsive UI

###  Admin Features

*  Secure Admin Login
*  Admin Dashboard for managing data
*  View user submissions (complaints/feedback)

###  System Features

*  Local data handling using DataStore
*  Modern Material Design UI
*  Fast and lightweight performance
